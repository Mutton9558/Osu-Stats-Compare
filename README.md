# osu! Stats Compare

<p>A website that compares users' osu! statistics. Through our site, users can easily compare their progress with another player!</p>
![Version](https://img.shields.io/github/package-json/v/mutton9558/osu-stats-compare)

# Features
1. Search players by their osu! usernames (including their previous ones!)
2. Side-by-side view of players' statistics
3. Displays:
    - Global and Country Rankings
    - PP Count
    - Accuracy
    - Play Count
    - Total Score
    - Play Time
    - Max Combo
    - Grades obtained
4. Both Mobile and Desktop view available!

# Future planned features 
1. Comparison features for other gamemodes (taiko, catch and mania) coming soon!!
2. osu! team comparison amongst members (coming in v2!!)
